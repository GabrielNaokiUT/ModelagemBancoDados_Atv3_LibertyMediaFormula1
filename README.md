# ModelagemBancoDados_Atv3_LibertyMediaFormula1

Modelagem de Banco de Dados - Atividade 3: Projeto Liberty Media Formula 1

A atividade consiste na modelagem e implementação de um banco de dados com **entidade única (Piloto)**, incluindo:

- MER (Modelo Entidade-Relacionamento)
- DER (Diagrama Entidade-Relacionamento)
- Implementação física no banco de dados

## 🧠 Tipo do projeto
Este trabalho representa um **CRUD básico**, ou seja:

- CREATE → Inserção de pilotos
- READ → Consultas dos dados
- UPDATE → Alteração das informações
- DELETE → Exclusão de registros

## 🏎️ Dados controlados
O sistema armazena informações como:

- Matrícula do piloto (identificador único)
- Nome completo
- País
- Idade
- Equipe
- Fabricante do motor
- Pontuação na temporada

## ⚙️ Tecnologias utilizadas
- PostgreSQL
- SQL
- pgAdmin 4
- Modelagem MER/DER

## 🎯 Objetivo
Demonstrar a implementação de um CRUD básico em banco de dados relacional, desde a modelagem conceitual até as operações de manipulação de dados.
